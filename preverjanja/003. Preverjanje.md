# 1. Lestev

Ustvari funkcijo `lestev` v kateri nariši lestev take oblike...

<img src="https://github.com/urosjarc/informatika/blob/main/media/turtle_lestev.png">

<details>

<summary>Prikazi pomoc</summary>

```python
import turtle

turtle.dot(10)
turtle.forward(100)
turtle.penup()
turtle.pendown()
turtle.left(20)
turtle.right(20)
turtle.speed(-1)
turtle.exitonclick()

def funkcija(a, b):
	return a + b

for i in range(0, 10, 1):
	print(i)

while(True):
	break

if(2 > 3):
	print("Hello")
```

</details>
