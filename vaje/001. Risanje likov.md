
# 1. Kvadrat

Ustvari funkcijo z imenom `kvadrat`,
ki sprejme celoštevilski argument `dolzina`.
Funkcija naj ob klicu nariše kvadrat z stranicami, ki so dolge
toliko kot je vrednost argumenta `dolzina`.


```python
kvadrat(100)
```
